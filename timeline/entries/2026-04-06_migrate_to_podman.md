Metron's server is migrated from Ubuntu to a new [CentOS](https://www.centos.org/) server, and the project is containerized to use [Podman](https://podman.io/). As part of this migration, [Jonas Stendahl](https://github.com/jyggen) joined the project as a system administrator and contributed to the infrastructure work.
