[MetronInfo](https://github.com/Metron-Project/metroninfo) v1.0, a new XML metadata schema for comic archives, is released as an open alternative to the existing ComicInfo.xml standard. The format improves on its predecessor with better data types, source tracking, and support for multiple comic databases.

[AJ Slater](https://github.com/ajslater), [Brian Pepple](https://github.com/bpepple), [Buried-In-Code](https://github.com/Buried-In-Code), and [Joe Milazzo](https://github.com/majora2007) all contributed to this schema.
