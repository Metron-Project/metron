What is Metron?

A community-built, open database for comic book metadata — free from corporate control.

Metron is a community-based site whose goal is to build an open database with a REST API for comic books.

Originally, when this project was created, the only comic book database that provided a REST API for users was Comic Vine. Unfortunately, it's a corporately-owned service and the community that uses it has no control over the resources provided to maintain it or ensure its continued accessibility. This project was started to create an alternative.

Note: Recently, the Grand Comics Database created an API, which you may wish to explore as an alternative.

What Metron Offers

Everything you need to discover, track, and organize your comic collection.

REST API

Access comic book metadata programmatically. Use the Python wrapper Mokkari to integrate Metron data into your projects with ease.

Create custom lists of issues to read, mark your progress, and share your reading order with others. Perfect for story arcs and event crossovers.

Wish List

Keep track of issues you want to acquire. Never lose track of the comics you're hunting for while shopping or trading with other collectors.

Pull List

Follow the series you love. Add ongoing titles to your pull list and get a consolidated view of everything you're currently collecting.

Quick Scrobble
