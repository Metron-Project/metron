{# What is Metron #}

What is Metron?

Metron is a community-based site whose goal is to build an open database with a REST API for comic books.

Originally, when this project was created, the only comic book database that provided a REST API for users was Comic Vine. Unfortunately, it's a corporately-owned service and the community that uses it has no control over the resources provided to maintain it or ensure its continued accessibility. This project was started to create an alternative.

Note: Recently, the Grand Comics Database created an API, which you may wish to explore as an alternative.

{# Using the REST API #}

Using the REST API

If you want to use data from Metron, there is a Python wrapper called Mokkari to make it easy to use the API.

Want to tag your comic archives with metadata from Metron? You can use these projects:

  • Metron-Tagger
  • Perdoo
  • Comic Tagger (No longer supported)

For more information on the API, please refer to the API Guidelines.

{# Want to Contribute #}

Want to Contribute?

If you would like to donate to help cover server and development costs, you can support us through OpenCollective.

If you want to add information to Metron, sign up for an account and read the Editing Guidelines before starting.

Want to help with the technical side of Metron? Metron is built with Django. If you have knowledge of Python and Django, head over to GitHub to check out the site's code.
